# Haohong Lai Academic Website

Static personal academic website for `https://haohong1.github.io/`.

## Structure

- `index.html`: page content and metadata
- `styles.css`: responsive visual design
- `script.js`: Chinese, English, and Spanish language switcher
- `assets/profile.jpg`: profile photo optimized for the web

## Deployment

The repository name for a GitHub Pages user site should be `haohong1.github.io`.
